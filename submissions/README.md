# 参加者のSkill提出場所

ここに自分が作成したSkillを提出してください。

## 提出方法

### Step 1: フォルダを作成
`submissions/あなたのGitHub名/スキル名/` フォルダを作成

```
submissions/
└── your-github-id/        ← あなたのGitHubユーザー名
    └── your-skill-name/   ← スキルの名前（英語推奨）
        └── SKILL.md       ← スキルファイル
```

### Step 2: SKILL.mdを作成
[テンプレート](../templates/SKILL-TEMPLATE.md) を参考に作成

### Step 3: Pull Requestを作成
変更をコミット・プッシュして、Pull Requestを作成

---

## 例

`examples/sample-user/greeting-skill/` に提出例があります。
参考にしてください。

---

## ルール

1. **自分のフォルダ内に作成する** - 他の人のフォルダは編集しない
2. **SKILL.mdを必ず含める** - これがないと動作しません
3. **descriptionにUse when:を含める** - AIが発動タイミングを判断するため
4. **個人情報を含めない** - 実名や学校名は記載しない

---

## 導入してもらう方法

あなたのSkillを他の人に使ってもらうには、以下のURLを共有：

```
https://github.com/School-Agent-Inc/orchestrate-it/tree/main/submissions/あなたのGitHub名/スキル名
```

相手はAntigravityで以下のように入力するだけ：

```
このリポジトリのskillを導入して
[上記のURL]
```
