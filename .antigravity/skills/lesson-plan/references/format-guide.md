## 2. 単元名

教科書の単元名または題材名を記載。
サブタイトルをつける場合もある。

```markdown
## 単元名

「分数のたし算とひき算」
〜分母が異なる分数の計算〜
```

---

## 3. 単元の目標

**必ず3観点で記述する。**

学習指導要領の目標に準拠し、この単元で育成を目指す資質・能力を明確にする。

```markdown
## 単元の目標

### 知識及び技能
- 異分母分数の加法及び減法の意味について理解している。
- 異分母分数の加法及び減法の計算ができる。

### 思考力・判断力・表現力等
- 分数の意味や表現を基に、異分母分数の加法及び減法の計算の仕方を考え、説明している。

### 学びに向かう力・人間性等
- 異分母分数の加法及び減法の計算について、数学的に表現・処理したことを振り返り、多面的に捉え検討してよりよいものを求めて粘り強く考えようとしている。
```

## 5. 単元設定の理由

3つの観点から、なぜこの単元をこのように指導するのかを説明する。

### 児童観（生徒観）
- クラスの実態
- 既習事項の定着状況
- 興味・関心の傾向
- 学習上の課題

### 教材観
- この単元の教育的価値
- 教科全体における位置づけ
- 既習事項との関連
- 今後の学習への発展

### 指導観
- 具体的な指導の方針
- 「主体的・対話的で深い学び」の実現に向けた工夫
- 個に応じた指導の工夫
- ICT活用等の手立て

```markdown
## 単元設定の理由

### 児童観

### 指導観
```

## 8. 本時の展開

45分（小学校）または50分（中学校・高等学校）の授業の流れ。

```markdown
## 本時の展開

| 時間 | 学習活動 | 指導上の留意点（◯）・評価（◆） |
|------|---------|------------------------------|
| **導入** 5分 | 1. 前時の振り返りをする<br>・同分母分数の加法の計算方法を確認する | ◯ 計算のポイントを想起させ、本時の学習につなげる |
| | 2. 本時の課題を確認する<br>【課題】1/2 + 1/3 の計算の仕方を考えよう | ◯ 分母が異なることに着目させ、課題意識を持たせる |
| **展開** 35分 | 3. 自力解決をする<br>・数直線や図を使って考える<br>・通分の考えを使う | ◯ 困っている児童には、「分母をそろえる」ことをヒントとして伝える<br>◯ 多様な考え方を認め、複数の解法を引き出す |
| | 4. グループで交流する<br>・自分の考えを説明する<br>・友達の考えと比較する | ◯ 「なぜその方法で計算できるのか」を説明し合うよう促す |
| | 5. 全体で共有する<br>・代表児童が発表する<br>・共通点や相違点を確認する | ◯ 通分の考え方に収束させる<br>◆ 分数の意味を基に計算の仕方を説明している【思①】 |
```

## 記述のポイント

### 学習活動の書き方
- 主語は児童・生徒
- 「〜する」の形で記述
- 具体的な活動内容を記載

### 指導上の留意点の書き方
- 主語は教師
- 「〜させる」「〜を促す」等の形で記述
- なぜその手立てが必要かを意識する

### 評価の書き方
- どの評価規準で評価するか明示（例：知①、思①）
- 評価方法を記載（ノート、発言、ワークシート等）
- 「おおむね満足できる」状態（B評価）を具体的に記述

---

## 「主体的・対話的で深い学び」の実現に向けて

### 主体的な学び
- 学習の見通しを持たせる
- 粘り強く取り組む場面を設定する
- 学習を振り返る活動を取り入れる

### 対話的な学び
- ペア・グループでの交流を設定する
- 多様な考えに触れる場面を設ける
- 先人の考え方（資料等）を活用する

### 深い学び
- 教科の「見方・考え方」を働かせる場面を設定する
- 知識を相互に関連付ける活動を取り入れる
- 問題を見いだし、解決する過程を重視する
