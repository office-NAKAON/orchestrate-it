## テンプレートの使い方

1. このファイルをコピーして、自分のスキルフォルダに配置
2. `name` と `description` を書き換える
3. **重要**: `description` に `Use when:` を必ず含める
4. 各セクションを自分のスキルに合わせて編集
5. 不要なセクションは削除してOK

### 配置場所
```
submissions/
└── あなたのGitHub名/
    └── あなたのスキル名/
        └── SKILL.md    ← このファイル
```

### descriptionの書き方のコツ
- 「〜を作成する」「〜を生成する」など具体的なアクション
- `Use when:` の後にトリガーワードを複数記載
- 日本語でOK

### 例
```yaml
description: |
  テスト問題を自動生成するスキル。
  Use when: テスト作成、問題作成、小テスト、確認テスト、定期テスト
```
